maven-site-plugin

Плагин maven-site-plugin предназначен для того чтобы легко создавать сайт проекта. В директории Вашего проекта наберите 

mvn site
И мавен автоматически сгенерирует сайт по нему. Сгенерированные файлы будут лежать в директории target/site.

Этот плагин можно довольно гибко настраивать, добавлять доплнитьные страницы, пункты меню и т.д. К примеру этот сайт создан с помощью этого плагина.
